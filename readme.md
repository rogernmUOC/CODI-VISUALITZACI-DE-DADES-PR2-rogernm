## Project overview
Projecte de visualització de dades sobre la relació entre vendes, valoracions
(crítics i usuaris) i engagement en videojocs.

## Repository contents
- NOTEBOOK.ipynb: processament i integració de dades
- final_dataset_for_flourish.csv: dataset final per a les visualitzacions
- steam_aggregated.csv: dades agregades de Steam (no inclós ja que pesa 2Gb)
- videogamesales.csv i metacritic_games_data.csv: datasets originals

## Visualisation
The interactive visualisation is available at:
